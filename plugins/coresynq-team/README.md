# coresynq-team

Claude Code plugin for working with Coresynq tasks in Notion.

Backed by a private MCP server on mac-mini. You need a `TEAM_KEY` from Collin to use it.

See `docs/install.md` for setup.

## What you get (depends on your role)

- **Noob:** add tasks for `coresynq` or `epcr`, see your own tasks, mark them done.
- **Co-dev:** read/write all Coresynq-domain tasks across the team.

You don't choose your role — your `TEAM_KEY` determines what's available. Run "who am I?" in chat to verify the install.
